EDF takes a file in which there should be a column - 'Ebay Item Number' and takes the data for api

Every 1000000 entries EDF creates a .csv file

The following data are taken from api:
Timestamp, ItemID, Product Title, Brand, SKU, Quantity, QuantitySold, Description, Categories, ConvertedCurrentPrice, OriginalRetailPrice,
ListingStatus, HitCount, UserId, StoreName, Pictures, ShippingServiceCost, ShippingType, ListedShippingServiceCost,
ItemCompatibilityCount, ManufacturerNumber, InterchangePartNumber, OtherPartNumber, Specifications,
Make, Model, Year, Trim, Engine, Notes
