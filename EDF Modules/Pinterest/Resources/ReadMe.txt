EDF required browser Chrome.
Move next files 'chromedriver.exe', 'WebDriver.dll' to /EasyDataFeed folder.

File with export should contains headers - 'Part Number', 'Brand', 'General Image', 'Spider URL', 'Description', 'Product Title'.
Flag 'Use existing export' - if unchecked 'Use Existing Export' EDF creates and downloads latest export but not save it on local pc. You can download export(it can be not actual) from backoffice by next steps:  Settings-> FTP -> feeds -> newSceExport.zip
Flag 'Use businesses account' - if you use business account, this flag should be checked.
