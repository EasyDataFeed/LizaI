EDF scrape  'Ebay Item Number' 'Sold' from Ebay stores.

For EDF operation it is necessary to write login and password from Ebay(seller) in Passwords - WebLogin

In result EDF creates .csv file on FTP .   name of file contains  timeStart_time_end_CountOfIds

example of file - 10-23-2017 21-03-55_10-23-2017 21-13-42_11146.csv
